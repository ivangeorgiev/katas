# Python Kata
Python Kata

# Install, Test, Build and Publish
## Install

```bash
$ git checkout https://github.com/ivangeorgiev/katas
$ cd katas
$ pip install -r requirements.txt
```

## Test
```bash
$ pytest
```

## Build

```bash
$ py -m build
```

## Publish
```bash
$ twine upload dist/*
```
